## 1. PitchBook API — Official Access Model

### PitchBook is a closed, enterprise-only platform

* **No free tier**
* **No developer sandbox**
* **No trial API keys**
* **No public endpoints**

Access requires:

* A **paid PitchBook subscription**
* Separate **API add-on contract**
* Legal agreement covering:

  * Data usage
  * Redistribution restrictions
  * Storage limits

Typical cost (order of magnitude):

* **Subscription**: high four to five figures annually
* **API add-on**: additional negotiated fee

There is **no workaround** that is legal, stable, or advisable.

## 2. What You Should NOT Do (Strongly Discouraged)

These approaches violate PitchBook’s Terms of Service and can expose you to legal and commercial risk:

* Scraping logged-in pages
* Automating browser sessions behind authentication
* Using shared or leaked credentials
* Reverse-engineering private endpoints
* Buying “PitchBook dumps” from third parties

For a professional workflow (especially investor-facing), this is non-negotiable.

## 3. Legitimate Ways to Get PitchBook Data at Low or Zero Cost

### Option A — Academic Access (Best “Free” Route)

* Enrolled at a university

* **UI access** (not API)

* Limited record counts

Limitations:

* No automation
* No programmatic API
* Usage restricted to academic purposes
